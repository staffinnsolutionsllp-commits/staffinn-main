## 1. PUBLIC RECRUITER PAGE FEATURES (/recruiter)
================================================================================

### HERO SECTION WITH SEARCH
**Feature Name:** "Recruiter Search Section"
**Background:** Uses recruiterbanner.jpg image with overlay
**Elements:**
- Page title: "Find Your Recruiter"
- Dual search functionality:
  - Recruiter name search input
  - Job title search input
- Search button with icon
- Real-time filtering of results

### RECRUITER LISTINGS GRID
**Feature Name:** "Company Profile Cards Grid"
**Data Source:** Real-time from DynamoDB users table + recruiter-profiles table
**Each Card Displays:**
- Company logo (80x80px) with fallback placeholder
- Company name with verified badge (✓)
- Industry information
- Open jobs count (dynamically calculated)
- "View Details" button
- Hover effects with transform and shadow

### INDIVIDUAL RECRUITER PROFILE PAGE
**URL Pattern:** /recruiter/:recruiterId
**Features:**
- Back navigation button
- Complete company profile display
- Profile photo with fallback
- Company information (name, industry, location, address, pincode)
- Recruiter details (name, designation, experience)
- Website link (if provided)
- Follow/Unfollow functionality (for staff users)
- Chat with recruiter button (opens Gmail)

### JOB LISTINGS SECTION
**Feature Name:** "Open Positions Display"
**Real-time Data:** Jobs filtered by recruiter ID
**Filter Options:**
- All Jobs
- Full-time
- Part-time
- Contract
- Freelance
- Internship
**Search Bar:** Real-time job search within recruiter's jobs
**Job Cards Display:**
- Job title and type badge
- Salary, experience, location
- Job description (truncated)
- Skills tags
- Posted date (formatted as "X days ago")
- Apply button with different states:
  - "Apply Now" (for staff)
  - "Apply Students" (for institutes)
  - "✓ Already Applied" (disabled state)

### COMPANY INSIGHTS SECTION
**Feature Name:** "About Company Information"
**Displays:**
- Company description
- Perks & benefits list (editable from dashboard)
- Hiring process steps (numbered workflow)
- Office images gallery (uploaded from dashboard)
- Common interview questions list

### REVIEWS SYSTEM
**Feature Name:** "Reviews & Insights"
**Functionality:**
- Add review form with star rating (1-5)
- Comment text area
- Review display with reviewer initials
- Show all/Show less toggle
- Real-time review submission and display

## 4. JOB MANAGEMENT SYSTEM
================================================================================

### JOB CREATION FORM
**Feature Name:** "Post New Job Modal"
**Required Fields:**
- Job Title
- Department (dropdown: Engineering, Product, Design, Marketing, Sales, Analytics, HR, Finance)
- Job Type (Full-time, Part-time, Contract, Freelance, Internship)
- Experience Required (format: "3-5" or "2")
- Salary Range
- Location
- Required Skills (comma-separated)
- Job Description

**Optional Fields:**
- Graduation Year
- Posted Date
- Status (Active/Closed/Draft)

### JOB LISTINGS TABLE
**Feature Name:** "Job Management Table"
**Columns:**
- Job Title
- Department
- Location
- Graduation Year
- Applications Count
- Status (with colored badges)
- Posted Date (formatted)
- Actions (Edit/Delete)

### JOB FILTERING SYSTEM
**Filter Options:**
- Search by title/department/location
- Status filter (All/Active/Closed/Draft)
- Department filter (All departments)

### JOB EDITING
**Feature Name:** "Edit Job Modal"
**Functionality:**
- Pre-populated form with existing data
- Same validation as creation
- Real-time updates to public listings
- Skills array handling

### JOB DELETION
**Feature Name:** "Job Removal System"
**Functionality:**
- Confirmation dialog
- Permanent deletion from database
- Automatic removal from public listings

## 10. TECHNICAL IMPLEMENTATION DETAILS
================================================================================

### DATABASE STRUCTURE
**Tables Used:**
1. **staffinn-users:** Basic user registration data
2. **recruiter-profiles:** Extended profile information
3. **staffinn-jobs:** Job postings
4. **staffinn-staff-profiles:** Staff applications
5. **job-applications:** Institute applications
6. **recruiter-news:** News management

### AUTHENTICATION & SECURITY
**Features:**
- JWT token authentication
- Role-based access control
- Password encryption
- File upload validation
- CSRF protection

### FILE UPLOAD SYSTEM
**Capabilities:**
- Profile photo upload (5MB limit)
- Office images upload
- News banner images
- Multer-based file handling
- Automatic filename generation

### REAL-TIME DATA SYNC
**Implementation:**
- Socket.io integration for visibility updates
- Real-time statistics updates
- Live profile changes
- Instant job posting updates

### RESPONSIVE DESIGN
**Features:**
- Mobile-responsive sidebar
- Adaptive grid layouts
- Touch-friendly interfaces
- Responsive tables with horizontal scroll

### API ENDPOINTS
**Public Endpoints:**
- GET /api/recruiter/public/all (All recruiters)
- GET /api/recruiter/public/:id (Individual recruiter)

**Private Endpoints:**
- GET /api/recruiter/profile (Get profile)
- PUT /api/recruiter/profile (Update profile)
- GET /api/recruiter/candidates (Get candidates)
- GET /api/recruiter/stats (Dashboard stats)
- POST /api/recruiter/upload-photo (Photo upload)
- POST /api/recruiter/:id/follow (Follow recruiter)

### DATA VALIDATION
**Validation Rules:**
- Email format validation
- Website URL validation
- File type and size validation
- Required field validation
- Experience format validation ("3-5" or "2")
