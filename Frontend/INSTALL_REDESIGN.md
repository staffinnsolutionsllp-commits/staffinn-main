# News Page Redesign Installation

## Required Dependencies

Run these commands in the Frontend directory:

```bash
npm install framer-motion sonner
```

## Fonts Setup

The Google Fonts (Playfair Display + DM Sans) have been added to index.html automatically.

## What's Been Redesigned

1. **NewsPage.jsx** - Complete UI/UX redesign with editorial magazine style
2. **NewsPage.css** - Premium design system with CSS variables
3. **index.html** - Added Google Fonts
4. **index.css** - Added global design tokens

## After Installation

1. Run `npm install` to install framer-motion and sonner
2. Start the dev server with `npm run dev`
3. Navigate to `/news` to see the redesigned page
