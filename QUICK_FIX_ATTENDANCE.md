# Quick Fix - Attendance Management

## Abhi Kya Karna Hai

### Step 1: Manual Attendance Use Karo
1. Attendance page pe jao
2. "Device Setup" button **IGNORE** karo (abhi kaam nahi karega)
3. **"Mark Attendance"** button click karo
4. Employee select karo
5. Check-in time enter karo (example: 09:00)
6. Check-out time enter karo (optional, example: 18:00)
7. Save karo

### Step 2: Employees Add Karo (Agar nahi hain)
1. **Onboarding** page pe jao
2. **"Add Candidate"** button click karo
3. Employee details fill karo:
   - Name
   - Email
   - Position
   - Department: "Onboarding" select karo
4. Save karo

### Step 3: Attendance Check Karo
1. Attendance page pe jao
2. Date select karo (calendar icon)
3. Attendance table mein employees dikhenge
4. Status check karo: Present/Late/Absent

## Troubleshooting

### "No employees found" dikha raha hai?
→ Pehle Onboarding page se employees add karo

### Attendance mark nahi ho raha?
→ Check karo:
- Employee ID correct hai?
- Check-in time valid hai?
- Internet connection hai?

### Stats show nahi ho rahe?
→ Page refresh karo (F5)
