## 8. POPULAR TAGS SYSTEM
================================================================================

### TAG GENERATION
**Feature Name:** "Popular Tags Cloud"
**Data Source:** Extracted keywords from all news articles
**Algorithm:** Keyword frequency analysis with common word filtering

**Tag Generation Process:**
1. Extract keywords from article titles and descriptions
2. Filter out common words (the, and, or, etc.)
3. Count keyword frequency across all articles
4. Weight search keywords higher (3x multiplier)
5. Sort by frequency and take top 15 tags
6. Display with frequency counts

### TAG DISPLAY
**Main Tags Section:** Above news feed with top 10 tags
**Sidebar Tags:** Complete tag cloud with all 15 tags
**Tag Format:** "TagName (count)" showing mention frequency
**Click Interaction:** Filter news by selected tag

### TAG FILTERING
**Filter Functionality:** Click tag to filter news articles
**Search Integration:** Adds clicked tags to search keywords
**Visual Feedback:** Smooth scroll to news feed on tag click
**Persistent Search:** Tags influence future popularity calculations

### TAG STYLING
**Visual Design:** Button-style tags with hover effects
**Count Display:** Small badge showing frequency count
**Responsive Layout:** Wrapping tag cloud for mobile devices

## 11. TECHNICAL IMPLEMENTATION DETAILS
================================================================================

### FRONTEND ARCHITECTURE
**Framework:** React with functional components and hooks
**State Management:** useState and useEffect hooks
**Styling:** CSS modules with responsive design
**Icons:** React Icons library (Font Awesome)

### BACKEND ARCHITECTURE
**API Endpoints:**
- `GET /api/news/all` - Get all news from all sources
- `GET /api/news/category/:category` - Get news by category
- `GET /api/news-admin/hero-sections/latest` - Get latest hero section
- `GET /api/news-admin/trending-topics/visible` - Get visible trending topics
- `GET /api/news-admin/expert-insights/visible` - Get visible expert insights
- `GET /api/news-admin/posted-news/visible` - Get visible posted news

### DATABASE STRUCTURE
**News Admin Tables:**
- `news-hero-sections` - Hero section content
- `news-trending-topics` - Trending topics data
- `news-expert-insights` - Expert insights and videos
- `staffinn-posted-news` - Posted news articles

**External News Tables:**
- `institute-events-news` - Institute events and news
- `recruiter-news` - Recruiter news and announcements

### FILE STORAGE
**S3 Bucket Organization:**
- `news-hero-sections/` - Hero banner images
- `news-trending-topics/` - Trending topic images
- `news-expert-insights-videos/` - Expert insight videos
- `news-expert-insights-thumbnails/` - Video thumbnails
- `staffinn-posted-news-photos/` - Posted news images
- `institute-events-news/` - Institute news images
- `recruiter-news/` - Recruiter news images

### AUTHENTICATION INTEGRATION
**Login Check:** isLoggedIn prop from parent component
**Login Modal:** onShowLogin callback for authentication
**Protected Features:** Full article reading, interactions, job alerts
**Public Access:** Basic news browsing and category filtering

### PERFORMANCE OPTIMIZATIONS
**Debounced Search:** 300ms delay for search operations
**Image Lazy Loading:** Placeholder images with error handling
**Efficient Filtering:** Client-side filtering for better performance
**Caching:** Browser caching for static assets

### ERROR HANDLING
**API Failures:** Graceful fallback to empty states
**Image Load Errors:** Automatic fallback to placeholder images
**Socket Disconnection:** Automatic reconnection attempts
**Loading States:** Visual feedback during data operations

### RESPONSIVE DESIGN
**Mobile Optimization:** Touch-friendly interfaces
**Breakpoint Handling:** Responsive grid layouts
**Image Scaling:** Adaptive image sizes for different screens
**Navigation:** Mobile-friendly carousel controls
